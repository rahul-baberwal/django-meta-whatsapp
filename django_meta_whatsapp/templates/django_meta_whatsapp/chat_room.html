Formatting Shortcuts

Basic Styling

  • Bold: Wrap text in asterisks *text* (Ctrl+B)
  • Italic: Wrap text in underscores _text_ (Ctrl+I)
  • Strikethrough: Wrap text in tildes ~text~ (Ctrl+U)
  • Monospace: Wrap text in triple backticks ```text```

Advanced Formatting (2024)

  • Bulleted Lists: Start a line with a hyphen and space - Item
  • Numbered Lists: Start a line with a number, period, and space 1. Item
  • Block Quotes: Start a line with a greater-than sign and space > Quote
  • Inline Code: Use single backticks `code`

Other Shortcuts

  • Send Message: Ctrl + Enter
  • Open Attachments: Ctrl + Shift + A
